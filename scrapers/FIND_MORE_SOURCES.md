# Finding More Event Sources for Toronto Kids

## Strategy 1: Google Search Operators

Use these search queries to find event calendars:

```
"Toronto" "kids events" "calendar"
"Toronto" "family events" site:.com
"GTA" "children's programs" "schedule"
"Toronto" inurl:events "kids" OR "children"
site:eventbrite.ca "Toronto" "kids" "free"
"Toronto" "drop-in" "children" "schedule"
```

**Advanced searches:**
```
"Toronto" kids events inurl:calendar
"Toronto" family activities site:*.ca
"GTA" children filetype:ics (finds calendar files)
```

## Validation Checklist

Before adding a source, verify:
- ✅ Has upcoming events (not just past)
- ✅ Updates regularly (weekly/monthly)
- ✅ Events are actually in Toronto/GTA
- ✅ Suitable for kids 0-12
- ✅ Has address/location info
- ✅ Website allows scraping (check robots.txt)
- ✅ Not requiring login to view events

## Priority Scoring

**High Priority:**
- Free events
- Regular schedule (weekly/daily)
- Multiple locations
- Clear event details
- Easy to scrape (HTML, not JavaScript)

**Medium Priority:**
- Mix of free/paid
- Monthly events
- Single location
- Some event details
- Moderate scraping difficulty

**Low Priority:**
- Mostly paid
- Occasional events
- Unclear details
- Difficult to scrape (JavaScript, login required)
